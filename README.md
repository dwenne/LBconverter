# SillyTavern ⇄ JanitorAI Lorebook Converter

A vibe-coded, browser-based converter for SillyTavern and JanitorAI lorebooks. No install, no account, runs entirely locally.

Supports:
- **ST World Info JSON ↔ JAI Lorebook JSON** (bidirectional)
- **JAI `loreEntries` scripts → ST** with category-based position mapping and scenario/personality splitting
- **JAI if-chain scripts → ST** (the simple `if (lastMessage.includes(...))` format)

ST export is basic JAI format only. No script generation.

## Usage

You can either download the latest version or use it [here](https://dwenne.github.io/LBconverter/lorebook-converter-v6.html) or [here](https://drevaine.neocities.org/lbconverter).
